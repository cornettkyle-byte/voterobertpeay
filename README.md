# Vote Robert Peay

Single-page campaign website for Robert Peay, County Commission District 4.

## Run

Open index.html in a browser. For live reload, use your preferred local web server.

## Build CSS (required for production)

1. Install dependencies: `npm install`
2. Build Tailwind: `npm run build`

## Customize

Replace placeholder text, endorsements, and map imagery with approved campaign content.
